How It Works

01

Find Empty Cell

Scan the grid left-to-right, top-to-bottom for the next unfilled cell (value = 0).

02

Try Digits 1–9

Attempt placing each digit and validate against row, column, and 3×3 box constraints.

03

Recurse

If valid, recurse deeper. If a dead end is reached, backtrack and try the next digit.

04

Complete

No empty cells left? Puzzle solved! Otherwise, report as unsolvable.

Time Complexity O(9^m) m = empty cells
Space Complexity O(m) recursion stack depth
